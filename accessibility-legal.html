Accessibility & Legal Information

Accessibility Statement

We are committed to making this website accessible to as many people as possible, regardless of ability or technology.

This site has been designed and tested with accessibility in mind, including:

  • Keyboard navigation support
  • Screen reader compatibility
  • Semantic HTML structure
  • Meaningful alternative text for non-decorative images
  • Clear focus indicators and logical focus order
  • ARIA attributes used only where appropriate

Some visual elements on this site rely on fixed positioning and precise layout to preserve the intended design experience. As a result, browser zoom and scaling may not behave in the same way as on more traditional, responsive layouts. Despite this, all essential content and functionality remains accessible via keyboard and assistive technologies.

We actively test this site using keyboard navigation and screen reader tools and make ongoing improvements as issues are identified.

If you encounter any accessibility barriers or have suggestions for improvement, please contact us and we will do our best to address them.

Use of Media

This website contains images, icons, and audio used to support the content and atmosphere of the site.

  • Images used for informational purposes include appropriate alternative text.
  • Decorative images are marked to be ignored by assistive technologies where applicable.
  • Audio playback is optional and user-controlled.

If you believe any media on this site has been used in error or without proper attribution, please contact us so the issue can be reviewed and corrected.
